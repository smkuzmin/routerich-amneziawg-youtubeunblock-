# Routerich: Настройка AmneziaWG и youtubeUnblock

Настраиваем на **[Routerich AX3000](https://routerich.ru/products/ax3000)** две WiFi-сети с разными способами обхода блокировок, чтобы не устанавливать VPN-клиенты на каждое устройство.

В результате настройки мы получим:

- Подключение по `WiFi 2.4 GHz`: весь Интернет идет через VPN-туннель **AmneziaWG**, что обеспечивает доступ ко всем заблокированным ресурсам (но с большей задержкой и меньшей скоростью)

- Подключение по `WiFi 5 GHz` или `LAN`: прямой доступ в Интернет, но с включенным **youtubeUnblock**, который позволяет смотреть заблокированный YouTube

**Характеристики настраиваемого WiFi-роутера:**

- Модель: `Routerich AX3000 v1`
- Архитектура: `ARMv8 Processor rev 4`
- Целевая платформа: `mediatek/filogic`
- Версия встроенного ПО: `RouteRich 24.10.5 r29087-d9c5716d1d RR-3.9.0 / LuCI openwrt-24.10 branch 26.039.68875~ec3d818`
- Версия ядра: `6.6.119`

Включите роутер, подключите кабель провайдера в порт `WAN`, а кабель от ПК в любой порт `LAN`. Затем откройте [Мастер настроек Routerich](http://192.168.1.1/luci-static/resources/wizard.html) (**Имя пользователя**: `root`, **Пароль**: *без пароля*) и пройдите все шаги мастера, заполняя соответствующие поля:

1. Интернет
   - **Протокол**: `PPPoE` (*обычно такой - смотрите свой договор на Интернет*)
   - **Имя пользователя**: *ваш логин из договора на Интернет*
   - **Пароль**: *ваш пароль из договора на Интернет*
   - **DNS-серверы**: `77.88.8.8`, `77.88.8.1`
   - **Включить IPv6**: `[ ]`
2. Сеть
   - **IP-адрес роутера**: `192.168.1.1`
   - **Маска сети**: `255.255.255.0`
3. WiFi
   - **Одинаковое имя для 2.4 и 5 ГГц**: `[ ]`
   - **Сеть 2.4 ГГц**: `RouteRich_2`
   - **Пароль**: `12345678`
   - **Сеть 5 ГГц**: `RouteRich_5`
   - **Пароль**: `12345678`
4. Mesh
   - **Включить Mesh-сеть**: `[ ]`

## Шаг 3: Настройка туннеля AmneziaWG

Откройте сайт: [WARP Генератор](https://warp-generator.github.io/) и в разделе **AmneziaWG** нажмите кнопку **AWG 2.0 (любой вариант)** и скачайте *файл конфигурации*. Затем в Web-интерфейсе роутера:

1. **Сеть** -> **Интерфейсы** -> **Добавить новый интерфейс..** -> **Имя**: `awg0`, **Протоколl**: `AmneziaWG VPN` -> **Создать интерфейс**
2. На вкладке **Основные настройки** -> **Импорт конфигурации**: нажмите **Загрузка конфигурации..** 
3. Откройте блокнотом ранее скачанный *файл конфигурации* (`WARP*.conf`) и вставьте его содержимое в пустое поле.
4. **Импорт настроек** -> **OK**
5. На вкладке **Настройки межсетевого экрана** -> **Создать / назначить зону межсетевого экрана**: `wan` -> **Сохранить**
6. Нажмите **Применить**

## Шаг 10: Настройка youtubeUnblock (исключение AWG)

в Web-интерфейсе роутера:

1. **Службы** -> **Терминал** -> **RouteRich login**: `root`
2. Выполните команды:
   ```bash
   nft insert rule inet fw4 youtubeUnblock oifname "awg0" return comment "skip_awg"
   nft list chain inet fw4 youtubeUnblock
   ```
3. Должно появиться правило `oifname "awg0" return comment "skip_awg"` в начале цепочки

## Шаг 11: Добавление скрипта автозапуска

в Web-интерфейсе роутера:

1. **Система** -> **Автозапуск**
2. На вкладке **Запуск пакетов и служб пользователя, при включении устройства** перед строкой `exit 0` вставьте:
   ```bash
   # Пропуск AWG-трафика через youtubeUnblock
   for i in $(seq 1 30); do
       if ip link show awg0 >/dev/null 2>&1; then
           sleep 5
           nft insert rule inet fw4 youtubeUnblock oifname "awg0" return comment "skip_awg" 2>/dev/null
           break
       fi
       sleep 2
   done
   ```
3. Нажмите **Сохранить**
4. **Система** -> **Перезагрузка** -> **Выполнить перезагрузку**

## Тестирование после перезагрузки

1. Подключитесь к `WiFi 2.4 GHz` (**Routerich_2**)
2. Проверьте IP-адрес - должен быть: `192.168.2.x`
3. Откройте [ifconfig.me](https://ifconfig.me) - должен быть IP AWG-сервера
4. Проверьте работу **[YouTube](https://www.youtube.com/)** и других заблокированных ресурсов, например **[X](https://x.com/)**
5. Подключитесь к `WiFi 5 GHz` (**Routerich_5**) или `LAN`
6. Проверьте IP-адрес - должен быть: `192.168.1.x`
7. Откройте [ifconfig.me](https://ifconfig.me) - должен быть IP вашего провайдера
8. Проверьте работу **[YouTube](https://www.youtube.com/)**

## Возможные проблемы и решения

### DHCP не выдает адреса на 2.4 GHz

1. Перезапустите **dnsmasq**: `/etc/init.d/dnsmasq restart`
2. Проверьте, что интерфейс **br-wifi24** поднят: `ip addr show br-wifi24`

### Интернет не работает на 2.4 GHz

1. Проверьте, поднят ли **AWG**: `ip link show awg0`
2. Проверьте правила маршрутизации: `ip rule show | grep 192.168.2`
3. Временно отключите **Firewall** для теста: `/etc/init.d/firewall stop`

### YouTube не работает на 5 GHz

1. Проверьте, работает ли **youtubeUnblock**: `logread | grep youtubeUnblock`
2. Убедитесь, что трафик не идет через **AWG**: [ifconfig.me](https://ifconfig.me) должен показывать IP провайдера

### Проверочные команды

```bash
# Проверка правил маршрутизации
ip rule show | grep 192.168.2

# Проверка таблицы 100
ip route show table 100

# Проверка правил youtubeUnblock
nft list chain inet fw4 youtubeUnblock

# Проверка зон Firewall
uci show firewall | grep "@zone" | grep "network"

# Проверка DHCP-аренд
cat /tmp/dhcp.leases | grep 192.168.2

# Проверка статуса AWG
ip link show awg0
```
